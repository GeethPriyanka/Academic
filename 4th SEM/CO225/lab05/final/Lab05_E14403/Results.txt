CONCLUSION

According to the Observations calculation time is minimum for integer type data and maximum for double type data. 
Effect of using threads can be observed clearly when the number of calculations are increased, because if the thread organizing time is greater than the calculation time then using threads might cause for an delay. Therefore, using threads is not going to make the program faster everytime. If the calculations and processes that have to be done in the program are greater, then threads can perform a great role to make the program faster. Number of threads that is going to make the most efficient calculation also depends on the program. However, in this case if the number of threads are equal to the number of rows then the performance is less.
